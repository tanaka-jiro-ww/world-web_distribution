## 再配布について
world webでの配布品は原則再配布禁止ですが、以下の場合は例外として許可しています。<br>
①単体のファイルを送信等することを主な目的としていない場合<br>
例：world webで配布されている画像等をScratch（プログラミング言語）で利用する（※）<br>
②初等教育、中等教育に関する目的の場合<br>
例：小中高校の授業での使用の場合（授業や家庭学習で使うプリント等に挿絵や補足資料等として載せる場合も可）なお、高等教育での使用は現状は禁止とします。［tarouni162@gmail.com］まで許可を得てください。<br>
③配布作品を用いた作品の共同作業の過程で共有する場合<br>
例：何らかのプロジェクトで使用する資料の作成を共同で行っている最中。なお、作業が完了した場合は二次的著作物として認められる場合は許可の申請は不要です。<br>
<br>
※例外として利用する場合も、出典元として当サイトへのリンクを明記してくださると幸いです。<br>
<br>
※Scratchは「リミックス」機能が再配布に該当しますが、ファイルの転送自体を目的としていないため例外として許可します。ただし、これはあくまでworld webの配布品にのみ適用されるルールです。他のサイトの素材を利用する際は、必ずそのサイトの規約を確認してください。
